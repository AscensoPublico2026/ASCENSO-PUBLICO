## PARTE A — Banco de conocimiento (formulario, 57 ítems)

**1. Word — Comparar documentos / control de cambios:** para ver los cambios hechos a un documento Word con control de cambios, la ruta recomendada es **REVISAR / Comparar / Comparar**.

**2. Word — Combinar correspondencia:** enviar un comunicado personalizado a 220 miembros usando una base de datos en Excel → función de **Correspondencia** (Combinar correspondencia / Mail Merge).

**3. Macros de Excel — finalidad:** **ejecutar de forma automática una acción o conjunto de acciones todas las veces que se desee** (automatizar tareas repetitivas).

**4. Tablas dinámicas — área de resumen:** el proceso descriptivo (Sumatoria, conteo, etc.) se configura desde el área de **Valores**.

**5. Excel — fijar columnas y encabezado al desplazarse:** herramienta **Inmovilizar paneles**.

**6. Word — pegar tabla manteniendo su formato original:** **Copiar manteniendo formato de origen**.

**7. PowerPoint — efectos al pasar de diapositiva:** **Transición de diapositiva**.

**8. PowerPoint — ver las diapositivas de forma panorámica/global:** vista **Clasificador de diapositivas** (botón de vistas correspondiente).

**9. PowerPoint — fondo con imagen original:** formato de fondo → apartado **Relleno** (relleno con imagen).

**10. Excel — etiquetar nivel de ingresos (bajo/medio/alto) según tabla:** función **=SI()** (anidada).

**11. Excel — unificar datos perdidos (NA, 8, 0 → N/A) eficientemente:** **Reemplazar todos**.

**12. Excel — administrar/analizar datos de una hoja, modificar formato y filtros especiales:** **Tablas dinámicas**.

**13. Windows — MAYÚS + SUPR:** **elimina permanentemente** el elemento (sin enviarlo a la papelera).

**14. Windows — CTRL + FLECHA DERECHA:** mueve el punto de inserción **al principio de la palabra siguiente**.

**15. Windows — ALT + ENTRAR (ALT + INTRO):** **ver las propiedades** del elemento seleccionado.

**16. Windows — ALT + F4:** **cerrar el elemento activo / salir del programa activo**.

**17. Windows — ALT + TAB:** **cambiar de un elemento abierto a otro** (cambiar de ventana).

**18. Windows — abrir el menú inicio con teclado:** **CTRL + ESC**.

**19. Windows — Windows + R:** aparece la pantalla **Ejecutar**.

**20. Word — CTRL + N:** coloca la palabra seleccionada en **negrita** (en Word en español, N = Negrita).

**21. Word — CTRL + D:** **alinear a la derecha** la porción de texto seleccionada.

**22. Word — CTRL + E:** **alineación centrada** (centrar) — ojo: en Windows español, CTRL+E también «selecciona todo»; en el contexto de la prueba la respuesta esperada es **alineación centrada**.

**23. Word — abrir un nuevo documento:** **CTRL + U**.

**24. Word — triple clic sobre una palabra:** **resalta/selecciona el párrafo completo** donde está la palabra.

**25. Word — subrayado verde ondulado:** indica **posibles errores gramaticales**.

**26. Excel — situarse en la última fila de un rango en una columna:** **tecla FIN + flecha abajo** (equivalente CTRL+flecha abajo).

**27. Excel — ir al último dato/última celda de la hoja:** **CTRL + tecla FIN**.

**28. Excel — tecla F1:** **abre la ayuda de Excel**.

**29. Excel — error `###`:** se soluciona **ampliando el ancho de la columna** para visualizar el resultado.

**30. Excel — doble clic en una celda:** **el cursor se fija dentro de la celda** (modo edición).

**31. PowerPoint — ver las diapositivas en modo presentación:** **F5**.

**32. PowerPoint — ALT + F4:** **cierra el programa**.

**33. PowerPoint — iniciar presentación desde la diapositiva actual:** **SHIFT + F5**.

**34. Outlook — abrir la libreta de direcciones:** **CTRL + SHIFT + B**.

**35. Outlook — eliminar el mensaje activo (sin eliminar la carpeta):** **CTRL + D**.

**36. Outlook — icono correo + flecha abajo:** **ordenar por importancia** (alta, media o baja).

**37. Outlook — al asignar tareas, además de aceptar y enviar informe de estatus:** se puede **marcar la tarea como completada**.

**38. Correo — documentos añadidos al cuerpo del mensaje:** **Archivos Adjuntos**.

**39. Outlook — evaluar mensajes según criterios y tomar acciones configuradas:** **Gestionar mensajes mediante reglas**.

**40. Correo — función de una carpeta:** **evitar que se acumulen mensajes sin relación en la bandeja de entrada** (organizar).

**41. Outlook — convocar a una reunión:** se usa cuando **una junta se llevará a cabo y usted es el organizador**.

**42. Outlook — publicar su progresión de actividad:** **una carpeta de red** (compartir disponibilidad/progreso).

**43. Outlook — elemento de la interfaz con árbol de menú para navegar bandejas y carpetas:** **Panel de navegación**.

**44. Correo — botón Reenviar:** **enviar el mensaje a otros destinatarios que no estaban incluidos** en el mensaje original.

**45. Outlook — delegado que solo puede leer elementos en las carpetas:** **Revisor**.

**46. Outlook — lugar donde están los archivos antes de enviarse (se remiten con F9):** **Bandeja de salida**.

**47. Outlook — seguimiento de una actividad hasta concluirla:** **crear una tarea**.

**48. Office — las fichas contextuales aparecen:** **cuando son necesarias**.

**49. PowerPoint — vista preliminar:** **ver la presentación antes de imprimir**.

**50. PowerPoint — guardar una presentación como plantilla:** **Botón Office → Guardar como → dar nombre y elegir tipo "plantilla"**.

**51. PowerPoint — ventaja de una presentación personalizada:** **incluir un subconjunto de diapositivas dirigido a una audiencia específica**.

**52. Office — CTRL + K:** activa **cursiva** (en el contexto de la prueba; nota: en Word español la cursiva suele ser CTRL+K).

**53. PowerPoint — ficha para modificar el formato del patrón de diapositivas:** **Diseño** (Vista → Patrón de diapositivas según versión).

**54. PowerPoint — grupo Edición de la ficha Inicio:** comandos **Buscar, Reemplazar y Seleccionar**.

**55. PowerPoint — movimiento del texto dentro de una diapositiva:** **Efecto** (animación del elemento).

**56. PowerPoint — ejecutar una presentación electrónica:** tecla **F5**.

**57. PowerPoint — la transición se aplica a:** **las diapositivas** (el paso entre diapositivas).

## PARTE B — Simulacro de Juicio Situacional (PJS) — 60 preguntas (formato oficial)

> Formato oficial de la PJS: no se memorizan atajos, se evalúa **cómo el candidato aplica la herramienta para resolver un problema administrativo real**. La opción correcta es la que aplica la herramienta adecuada/buena práctica.

### Word
- Enviar carta de notificación a 150 aspirantes (plantilla Word + lista en Excel) → **Combinar Correspondencia (Mail Merge) vinculada al Excel**.
- Manual de 80 páginas con logo en encabezado excepto la portada → **insertar logo en el encabezado y marcar "Primera página diferente"**.
- Tabla de contenido (índice) automática en informe de 80 páginas → **aplicar Estilos de título (Título 1, Título 2…)**.
- Equipo jurídico revisa borrador sin modificar el original → **Control de Cambios / Insertar Comentarios (Track Changes)**.
- Pegar texto de web sin formato extraño → **Pegar → "Mantener solo texto" (Keep Text Only)**.
- Numeración i, ii, iii en la introducción y reiniciar en 1, 2, 3 desde el Capítulo 1 → **Salto de Sección antes del Capítulo 1 y desvincular la numeración**.
- Distribución final inalterable y legible en cualquier dispositivo → **formato .PDF**.
- Índice alfabético de términos (glosario) con páginas → **"Marcar entrada" e "Insertar Índice"**.
- Una página horizontal (tabla grande) entre páginas verticales → **Salto de Sección antes y después y cambiar la orientación de esa sección**.
- Citar fuente y gestionar bibliografía → **"Insertar Nota al Pie" / "Insertar Cita"**.
- Número de página que se actualiza solo → **Insertar → Número de página**.
- Buscar "Contrato" y cambiar por "Acuerdo" en todo el documento → **Buscar y Reemplazar (Ctrl+L / Ctrl+B)**.

### Excel
- Contar registros (celdas no vacías) en columna Radicado → **=CONTARA(A2:A5001)**.
- Mostrar solo PQR Tipo="Queja" y Estado="Abierto" sin borrar el resto → **Filtros (AutoFilter)**.
- Estandarizar "Medellín", "medellin ", " MEDELLIN" a un solo término → **Buscar y Reemplazar**.
- Resumen de ventas por Región desglosado por Vendedor → **Tabla Dinámica (Pivot Table)**.
- Ver registros de fecha más antiguos primero → **Ordenar A→Z (ascendente)**.
- Fijar encabezados al desplazarse a la fila 2000 → **Vista → Inmovilizar Paneles (Freeze Panes)**.
- Mostrar proporción/partes de un todo → **gráfico Circular (Pie)**; tendencia de quejas ene-dic → **gráfico de Línea**.
- `$I$1` en una fórmula → **Referencia Absoluta** (no cambia al copiar la fórmula).
- Comisión 10% si ventas > 100 → **=SI(A2>100; B2*0,1; 0)**.
- Traer Salario de Hoja1 a Hoja2 usando Cédula como llave → **=BUSCARV (VLOOKUP)**.
- Sumar solo contratos de "Secretaría de Obras" → **=SUMAR.SI(A:A; "Secretaría de Obras"; B:B)**.
- Contar empleados de "Secretaría de Tesorería" → **=CONTAR.SI(A:A; "Secretaría de Tesorería")**.
- Clasificar por cargo (Profesional/Técnico…) → **SI anidado**.
- Valor total = precio × cantidad en D2 → **=B2*C2**.
- Tabla dinámica muestra "Suma de Cédula" en vez de cantidad → **cambiar Configuración del campo de valor de "Suma" a "Cuenta"**.
- Resaltar en rojo valores < 1000 → **Formato Condicional**.
- Separar nombre y apellidos de una columna → **Texto en Columnas**.
- Lista desplegable con opciones predefinidas (Abierto/Cerrado/En Trámite) → **Validación de Datos**.
- Unir "Ana" + "Pérez" en una celda → **=CONCATENAR(A2;" ";B2)** o **A2&" "&B2**.
- =PROMEDIO con filtro que oculta filas → **la fórmula sigue promediando las 10 filas (ignora el filtro)**; para respetar el filtro usar **SUBTOTALES** (suma/promedia/cuenta excluyendo filas ocultas por filtro).
- =SI(A1>10;"Aprobado";"Rechazado") con A1=10 → muestra **"Rechazado"** (10 no es > 10).

### PowerPoint
- Logo en todas las diapositivas (nuevas y existentes) → **Patrón de Diapositivas (Slide Master)**.
- Diapositiva con 20 líneas en fuente 8 → error porque **el texto es muy pequeño; la diapositiva es apoyo visual, no un documento**.
- Organigrama de la entidad → **SmartArt tipo Jerarquía**.
- Animaciones llamativas en rendición de cuentas → evitar porque **distraen del mensaje y restan seriedad**.
- Revelar 5 puntos uno por uno al hablar → **Animación ("Aparecer") por viñeta, "Al hacer clic"**.
- Diferencia Transición vs Animación → **Transición = cómo se pasa entre diapositivas; Animación = cómo aparecen los elementos dentro de una diapositiva**.
- Enlace web al PDF al final → **Insertar → Hipervínculo**.
- Imagen tapa el título → **clic derecho → "Enviar al Fondo"**.
- Modo Presentador → **el presentador ve sus notas y la siguiente diapositiva; el público solo la actual**.
- Patrón de Diapositivas (propósito) → **establecer diseño, fuentes y colores predeterminados de toda la presentación**.
- SmartArt → **crear diagramas visuales (listas, procesos, ciclos, jerarquías)**.

### Outlook / correo y seguridad
- Correo del "Banco de Occidente" pidiendo "verificar cuenta" → **Phishing (suplantación)**; acción: **no hacer clic, verificar por canal oficial, no enviar credenciales**.
- "Soporte Técnico" pide credenciales → **verificar llamando a la mesa de ayuda / TI por canal oficial; no responder ni hacer clic**.
- Informe al Jefe (Para), informar a Secretario y Líder → **Para: Jefe; CC: Secretario, Líder**.
- Invitación a 200 ciudadanos protegiendo privacidad → **campo CCO (Con Copia Oculta / BCC)**.
- Diferencia CC vs CCO → **los de CC son visibles para todos; los de CCO son invisibles para los demás**.
- Reenviar → **enviar el mensaje a destinatarios que no estaban en el original**.

### Gestión de archivos, nube y colaboración
- Múltiples versiones "Informe_FINAL_FINAL.docx" → problema: **falta un sistema de control de versiones**; solución: **nube (OneDrive/SharePoint) con coautoría en tiempo real**.
- Compartir carpeta de 500 MB con asesor externo → **subirla a la nube y enviar el enlace de descarga**.
- Archivo ".zip" → **archivo/carpeta comprimido para reducir peso y facilitar el envío**.
- Antivirus detecta amenaza en adjunto → **poner en cuarentena/eliminar según el antivirus y NO abrirlo**.
- Guardar vs Guardar Como → **Guardar actualiza el archivo existente; Guardar Como crea una copia nueva (otro nombre/ubicación)**.
- Coautoría en OneDrive/SharePoint → **edición simultánea del mismo archivo**.

### Atajos transversales
- Deshacer la última acción → **Ctrl + Z**.
