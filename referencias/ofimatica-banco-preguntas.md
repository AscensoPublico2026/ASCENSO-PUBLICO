## Cobertura temática detectada (mapa de la prueba)

La prueba combina **dos enfoques**:

1. **Juicio situacional (PJS):** "¿qué herramienta usar para resolver este problema
   administrativo real?" — el enfoque oficial de la CNSC.
2. **Conocimiento funcional / atajos:** qué hace una función o una combinación de teclas.

### Microsoft Word (procesador de texto)
- Control de cambios y revisión: ver/seguir cambios → **Revisar / Seguimiento / Panel de revisiones**.
- **Combinar correspondencia** (Mail Merge) con base de datos de Excel (cartas masivas personalizadas).
- **Pegado especial**: mantener formato de origen / de destino / solo texto / como imagen.
- **Encabezado y pie de página**; logo en todas las páginas excepto la portada → "Primera página diferente".
- **Tabla de contenido (índice) automática** a partir de **Estilos** de título.
- **Saltos de sección**: numeración i, ii, iii y reinicio en 1, 2, 3; orientación mixta (vertical/horizontal).
- **Índice alfabético / glosario**: "Marcar entrada" + "Insertar índice".
- **Notas al pie** y **citas / bibliografía**.
- **Buscar y reemplazar**.
- **PDF** como formato de distribución final (no modificable, legible en cualquier dispositivo).
- Atajos Word: CTRL+N (negrita), CTRL+K (cursiva), CTRL+S (subrayado), CTRL+U (nuevo documento),
  CTRL+D (cuadro Fuente), CTRL+E (centrar), triple clic (seleccionar párrafo),
  subrayado verde ondulado (error gramatical), CTRL+Z (deshacer), CTRL+Y (rehacer).

### Microsoft Excel (hoja de cálculo)
- **Macros**: automatizar acciones repetitivas.
- **Tablas dinámicas**: área de **Valores** define la operación (suma, conteo…); "Configuración de
  campo de valor" para cambiar de Suma a Cuenta.
- **Inmovilizar paneles** (fijar encabezados/columnas al desplazarse).
- **Filtros** (AutoFilter) y **Ordenar** (A-Z / Z-A, fechas, etc.).
- **Buscar y reemplazar** / "Reemplazar todos" para estandarizar datos.
- **Insertar tabla** (rango con formato + filtros + herramientas especiales).
- **Validación de datos** (listas desplegables).
- **Texto en columnas** (separar nombre/apellidos).
- **Formato condicional** (resaltar según condición).
- **SUBTOTALES** (respeta el filtro, a diferencia de SUMA).
- Fórmulas: **SI** (y SI anidada), **BUSCARV**, **SUMAR.SI**, **CONTAR.SI**, **CONTARA**,
  **CONCATENAR** (o &), **PROMEDIO**, multiplicación (=B2*C2).
- **Referencias absolutas** ($) vs relativas.
- **Gráficos**: línea (tendencia en el tiempo), circular/anillos (proporción), dispersión.
- Error **###** → ampliar el ancho de columna.
- Atajos Excel: FIN+flecha / CTRL+flecha (saltar al extremo de datos), CTRL+FIN (última celda usada),
  F1 (ayuda), F2 / doble clic (editar celda).

### Microsoft PowerPoint (presentaciones)
- **Patrón de diapositivas** (Slide Master): logo, fuentes y colores para toda la presentación.
- **Transición** (entre diapositivas) vs **Animación** (elementos dentro de una diapositiva).
- Animación "Al hacer clic" para revelar viñetas una por una.
- **SmartArt** (jerarquías/organigramas, procesos, ciclos, listas).
- **Vistas**: clasificador de diapositivas (vista panorámica del orden).
- **Formato de fondo → Relleno** (imagen de fondo).
- **Hipervínculo** (enlace a web/PDF).
- **Enviar al fondo** (imagen que tapa el título).
- **Modo presentador** (ver notas y la siguiente diapositiva).
- **Guardar como plantilla**; presentación personalizada (subconjunto de diapositivas).
- Buenas prácticas: apoyo visual (no documento), evitar texto diminuto y animaciones que distraen.
- Atajos: F5 (presentar desde el inicio), SHIFT+F5 (desde la diapositiva actual), ALT+F4 (cerrar),
  CTRL+K (cursiva).

### Microsoft Outlook / correo electrónico
- Campos **Para / CC / CCO (BCC)**: CC visible para todos; **CCO oculto** (privacidad de listas masivas).
- **Archivos adjuntos**.
- **Gestionar mensajes mediante reglas** (acciones automáticas según criterios).
- **Carpetas** (evitar acumulación en la bandeja de entrada).
- **Reenviar** vs Responder.
- **Tareas** (crear/seguir hasta concluir; marcar como completada).
- **Convocar una reunión** (usted es el organizador).
- **Panel de navegación** (árbol de bandejas y carpetas).
- **Permisos de delegado**: Revisor (solo lectura), Autor, Editor.
- **Bandejas**: salida (F9 = enviar/recibir), entrada, borradores, no deseado, papelera.
- **Libreta de direcciones**: CTRL+SHIFT+B. **Eliminar mensaje**: CTRL+D.

### Windows y atajos del sistema
- MAYÚS+SUPR (eliminar permanentemente, sin papelera).
- CTRL+→ (inicio de la palabra siguiente).
- ALT+ENTRAR (propiedades del elemento).
- ALT+F4 (cerrar elemento activo / salir del programa).
- ALT+TAB (cambiar entre elementos abiertos).
- CTRL+ESC (abrir menú Inicio).
- Windows+R (cuadro Ejecutar).

### Seguridad informática y buenas prácticas
- **Phishing** (correos que suplantan identidad para robar credenciales) → no hacer clic, verificar por canal oficial.
- **Antivirus**: poner en cuarentena/eliminar, nunca abrir el adjunto sospechoso.
- **Archivos .zip** (carpetas comprimidas para reducir peso y facilitar el envío).
- **Compartir archivos pesados** vía nube (OneDrive, WeTransfer, SharePoint) + enlace, no por adjunto.
- **Control de versiones / coautoría** en la nube (evitar "Informe_FINAL_FINAL_v3").
- **Guardar** vs **Guardar como** (actualizar vs crear copia nueva).
- Copias de seguridad y protección de datos personales (Ley 1581 de 2012).
